A local multiplayer social deduction party game built in Python, featuring a Large Language Model (LLM) as the ultimate Game Master. Inspired by viral "pass-the-phone" games like *Undercover*, but dynamically generated by AI.


## 🎮 How It Works

Gather 3 or more friends around a single terminal. Players select a topic (e.g., "Soccer Players", "Fast Food Chains"), and the AI secretly assigns everyone the same word—except for one randomly chosen **Impostor**, who receives a closely related trick word. 

1. **Pass & Play:** Players secretly check their roles and words on the terminal.
2. **Give Clues:** Everyone says a one-word clue out loud to prove they know the true word.
3. **Vote:** The group votes on who they think the Impostor is.
4. **The Twist:** If the Impostor is caught, they get one final chance to steal the victory by guessing the Survivors' secret word. The AI acts as the judge!

## ✨ Features

* **Infinite Replayability:** No pre-made word lists. The AI generates custom pairs of words on the fly based on *any* user-inputted topic.
* **Smart UI/UX:** Built-in terminal clearing and pause mechanisms ensure secret roles remain hidden during device-passing.
* **Dynamic State Management:** Uses Python dictionaries to dynamically track active players, count votes, handle ties, and update the game state round-by-round.
* **AI Judgment System:** Evaluates the Impostor's final guess for semantic closeness rather than just requiring an exact text match.
